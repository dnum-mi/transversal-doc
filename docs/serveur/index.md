# Les technologies Serveur de la Fabrique Numérique

Les projets serveur de la Fabrique Numérique utilisent principalement [Fastify](https://www.fastify.io/) et [NestJS](https://nestjs.com/) pour l'écosystème Node.js/TypeScript, ainsi que [FastAPI](https://fastapi.tiangolo.com/) pour les projets Python.

## [NestJS](/serveur/nestjs)

Conventions et bonnes pratiques pour les projets NestJS : nommage, structure, logging, gestion des erreurs, documentation OpenAPI.

## [Fastify](/serveur/fastify)

Conventions et bonnes pratiques pour les projets Fastify : plugins, validation, logging, gestion des erreurs.

## [FastAPI](/serveur/fastapi)

Conventions et bonnes pratiques pour les projets FastAPI : structure, Pydantic, endpoints asynchrones.

# Les fondamentaux de la CI/CD

L'intégration continue (CI) et le déploiement continu (CD) sont des pratiques essentielles pour garantir la qualité et la fiabilité des projets de la Fabrique Numérique.

## [Les principes de la CI/CD](/ci-cd/principes)

Comprendre les concepts fondamentaux, les outils (GitHub Actions, SonarQube, Trivy) et les différentes phases d'un pipeline CI/CD, avec des exemples simples pour démarrer.

## [Workflows réutilisables](/ci-cd/workflows-reutilisables)

::: tip Approche recommandée
La Fabrique Numérique maintient des **workflows GitHub Actions réutilisables** dans le dépôt [`dnum-mi/fabnum-cicd`](https://github.com/dnum-mi/fabnum-cicd). Privilégiez ces workflows prêts à l'emploi pour vos pipelines CI/CD.
:::
